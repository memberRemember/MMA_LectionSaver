Для корректной работы нужен Python 3.12.7

**Зависимости**
- Python 3.12.7  
- [Playwright](https://playwright.dev/python/)  
- [PyPDF](https://pypdf.readthedocs.io/en/latest/)  

## Работа скрипта
При запуске скрипт запросит название курса, с которого нужно начать скачивать лекции, и номер лекции.

Если название курса оставить пустым, то будут скачаны дисциплины по порядку.

**Номер лекции нужно ввести обязательно.**
Лекции скачиваются в папку `/pdf`. Изменить папку можно в переменной `OUTPUT_DIR = "{путь вашей папки}"`
Скрипт скачивает все лекции отдельно, затем объединяет их в файл `<Название_курса_ALL.pdf>`

Уже скачанные лекции пропускаются.

### Опционально
По-умолчанию скрипт открывает окно браузера и выполняется.

Можно изменить это поведение параметром `headless = True`, и смотреть за процессом в терминале:
```
browser = p.chromium.launch(headless=False) # открывает браузер
```

```
browser = p.chromium.launch(headless=True) # работает тихо
```

## Запуск
**1. Скачать репозиторий или склонировать его**

**2. Создать виртуальное окружение с версией Python 3.12.7 и активировать его**

macOS / Linux:
```
python -m venv env
source env/bin/activate
```

Windows:
```
python -m venv env
env\Scripts\activate
```

**3. Установить зависимости**
```
pip install -r requirements.txt
```

**4. Установить браузер для Playwright**
```
playwright install chromium
```

**5. Установить Ваш логин и пароль**
> [!IMPORTANT]
> Вставьте Ваш логин в переменную `EMAIL = "Ваш логин"`\
> Вставьте Ваш пароль в переменную `PASSWORD = "Ваш пароль"`

**6. Запустить скрипт**
```
python lectionsaver.py
```
